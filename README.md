# Healthcare-Data-Analysis (Interactive Dashboard)
## Project Objective:
Compare the average insurance cost for the year 2023.

Compare medical diagnosis between female and male patients.

## Dataset used:
- <a href="https://github.com/keilacodes/Healthcare-Data-Analysis/blob/main/healthcarevisualdashboard.xlsx">Dataset</a>

## Questions (KPIs)
Compare cost of different insurance companies.
Analyze each insurance provider by quarter.
Which annual quarter had the highest cost?
What was the total cost of insurance combined?
What was the average annual cost of insurance?

Compare medical diagnosis between genders.
Relation between gender and medical conditions.
What amount of patients show positive results?
Which results are highest for each gender?

- Dashboard Interaction <a href="https://github.com/keilacodes/Healthcare-Data-Analysis/blob/main/InsuranceAverage.png"> View Insurance Dashbaord</a>

- Dashboard Interaction <a href="https://github.com/keilacodes/Healthcare-Data-Analysis/blob/main/MedicalConditions.png"> View Medical Dashbaord</a>

## Process

Verify data for any missing values and anomalies, and sort out relevant information.
Ensure data is consistent and clean with respect to the data type, data format and values used.
Created pivot tables according to data visualization depicted.
Merge pivot tables into the dashboard and applied slicer to make dynamic.
